# Automatic Number Plate Recognition (ANPR) Project

This project detects Indian vehicle number plates from images using EasyOCR and OpenCV.

## How to use

1. Clone the repository:
git clone <your-repo-link>
cd ANPR_project

2. Install dependencies:
pip install -r requirements.txt

3. Open `anpr.py` and set your image path. Then run:
python anpr.py

Detected number plates will be displayed and saved to an output file.
